## Project Description

This project predicts whether a patient has heart disease using a Machine Learning model (Logistic Regression) and a Streamlit web application.

---

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Joblib
- Streamlit

---

## Installation

Install the required libraries:

```bash
python -m pip install -r requirements.txt
```

---

## Steps to Run the Project

### Step 1: Train the Model

```bash
python train_model.py
```

### Step 2: Run the Streamlit Application

```bash
python -m streamlit run app.py
```

Open your browser and visit:

```
http://localhost:8501
```

---

## Features

- Load Heart Disease dataset
- Data preprocessing using One-Hot Encoding
- Train Logistic Regression model
- Make predictions
- Display Confusion Matrix
- Calculate Accuracy, Precision, Recall, and F1 Score
- Save and Load trained model using Joblib
- Interactive Streamlit Web Application for Heart Disease Prediction
